## Описание

Данный проект содержит реализации функций для работы с прямоугольниками и кругами, а также модульные тесты, созданные с помощью стандартного модуля Python `unittest`.

Работа демонстрирует написание unit-тестов, проверку корректности функций, работу с ошибочными входными данными и простую проверку производительности.

---

## Функциональность проекта
  
### Функции для прямоугольника:
- `rect_area(a, b)` — площадь  
- `rect_perimeter(a, b)` — периметр  
- `rect_diagonal(a, b)` — диагональ  
- `rect_scale(a, b, k)` — масштабирование  

### Функции для круга:
- `circle_area(r)` — площадь круга  
- `circle_circumference(r)` — длина окружности  
- `circle_diameter(r)` — диаметр  
- `circle_is_unit(r)` — проверка единичного круга  
- `circle_scale(r, k)` — масштабирование
